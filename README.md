# Shiksharthi Website

Official website for **Shiksharthi Educational Institute**.

This project is a modern, responsive website built to showcase the institute, its academic programs, student achievements, and provide access to important learning resources.

---

## 🌐 Live Website

https://shiksharthi.org

---

## 🚀 Features

* Modern responsive design
* Student achievers section
* Academic program information
* Fast and optimized performance
* Clean UI built with modern frontend technologies

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/aryya-dev/shiksharthi-website.git
```

Navigate into the project directory:

```
cd shiksharthi-website
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

The website will run locally at:

```
http://localhost:5173
```

---

## 🏗 Build for Production

To create an optimized production build:

```
npm run build
```

The production files will be generated in the `dist` folder.
