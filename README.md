## ⚙️ Installation

Clone the repository

    git clone https://github.com/aryya-dev/shiksharthi-website.git

Navigate to project folder

    cd shiksharthi-website

Install dependencies

    npm install

Run development server

    npm run dev

The website will run locally at

    http://localhost:5173

---

## 🏗 Build for Production

Create an optimized production build

    npm run build

Production files will be generated in

    dist/
