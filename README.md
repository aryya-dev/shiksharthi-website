# Shiksharthi Website

Official website for **Shiksharthi Educational Institute**.

This project is a modern responsive website built to showcase the institute, its programs, student achievements, and provide access to academic resources.

## 🌐 Live Website
https://shiksharthi.org

## 🚀 Features

- Responsive modern UI
- Student achievers section
- Program information pages
- Academic resources
- Admin / portal interface
- Fast performance with Vite

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
